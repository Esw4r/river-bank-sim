
# River Bank Simulator

a fun project heavily inspired by dwarf fortress.


## play the game

clone the project

```bash
  git clone https://github.com/s4nj1th/river-bank-sim
```

go to the project directory

```bash
  cd river-bank-sim
```

compile the code

```bash
  g++ main.cpp -o main
```

run the game

```bash
  ./main
```
